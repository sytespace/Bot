:unlock: User Commands:

× `s!ping` - Gets the bot's latency
× `s!skin <minecraft username>` - Renders a minecraft skin
× `s!new` - Opens a new ticket
× `s!pfp [@user]` - Gets a user's profile picture, if you don't specify a user it will get your profile picture
× `s!tos` - Gets the bot's Terms of Service
× `s!serverinfo` - Gets some info of the server
× `s!uptime` - Gets the bot's uptime
× `s!genpw` - Genarates a secure password and DM's it to you
× s!close - Closes the current ticket
× `s!profile [@user]` - Gets a user's profile, if you don't specify a user it will get your profile
